# LLM-App

  ## Description
  This project combines Bun, Tailwind CSS, shadcn/ui, Prisma, and more to create two uses for LLM models. A chatbot that responds to inquiries(park hours, ticket information, etc..), for a fictional amusement park offering park information. And summarizer that takes customer reviews for products and gives a summarization of the reviews from the backend.

  ## Installation
This project was created using `bun init` in bun v1.3.2. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.  

  To start using this code follow these instructions:

  To install dependencies:

```bash
bun install
```


  ## Usage

The LLM of your choosing can be used ran either locally or via a service.  

A backend service, for which i used MySQL, is needed to pull data from for the summarizer.  

To run:

```bash
bun run index.ts
```
